# Personal Expense Tracker (Python, JSON)

A small command-line personal expense tracker written in Python.
Stores expenses in a JSON file (`expenses.json`) and supports:
- Add an expense (amount, category, note, date)
- View all expenses (sorted by date)
- View total spending and spending by category
- Search by category/note/date fragment
- Delete an expense by ID

## Requirements
- Python 3.7+

## How to run
1. Clone or download this repo
2. Ensure `expense_tracker.py` is executable or run:
   ```bash
   python3 expense_tracker.py
